# 🩺 AI Disease Prediction System

An AI-powered web application that predicts possible diseases based on user-selected symptoms using a Machine Learning model. The application is built with **Python**, **Flask**, and **Scikit-learn**, providing a simple and interactive interface for educational purposes.

---

## 🚀 Features

- 🔍 Predicts diseases based on selected symptoms
- 🤖 Machine Learning-based prediction
- 💊 Displays health precautions
- 🥗 Suggests a recommended diet
- 👨‍⚕️ Provides basic doctor advice
- 📊 Shows prediction confidence
- 💻 Responsive and user-friendly interface

---

## 🛠️ Technologies Used

- Python
- Flask
- Pandas
- Scikit-learn
- Joblib
- HTML5
- CSS3

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/khadija1407/Disease-Prediction-System.git
```

### 2. Navigate to the project folder

```bash
cd Disease-Prediction-System
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Train the Machine Learning model

```bash
python train_model.py
```

### 5. Run the Flask application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```

---

## 📷 Application Workflow

1. Select symptoms from the list.
2. Click **Predict Disease**.
3. The system predicts the disease using the trained ML model.
4. View precautions, diet recommendations, doctor advice, and prediction confidence.
